# Default Meme Templates

Place your JPG image files in this directory to use them as default templates in the meme generator.

## How to add templates:

1. Add your JPG images to this `images` folder
2. Name them: `template1.jpg`, `template2.jpg`, `template3.jpg`, etc.
3. The meme generator will automatically detect and display them

## Supported formats:
- JPG/JPEG files
- PNG files
- Other common image formats

## Notes:
- Images will be displayed in a grid gallery
- Click on any template to use it
- You can still upload your own images using the "Upload Your Own Image" button
